# VidSave – Video Downloader

A simple, fast web app to download videos from YouTube, Instagram, TikTok, Threads, and 1000+ other platforms.

## Features

- 🎥 Download from 1000+ video platforms
- 🎨 Modern dark UI with real-time progress tracking
- ⚡ Multiple quality options (1080p, 720p, Best MP4, Audio)
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎭 Support for playlists (downloads as ZIP)
- 🧵 Custom Threads scraper support

## Installation & Setup

### Prerequisites
- Python 3.8+
- Google Chrome (for Threads support)

### Quick Start

1. Clone and navigate to the folder:
   ```bash
   git clone <repo-url>
   cd VidSave
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   python app.py
   ```

4. Open your browser to `http://localhost:5000`

## How to Use

1. Paste a video URL into the input field
2. Select desired quality (1080p, 720p, Best, or Audio only)
3. Click **Download**
4. Watch the progress bar and bandwidth stats
5. Your file downloads automatically
